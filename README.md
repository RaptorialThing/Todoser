# Todoser

### Requirements

`ruby >= 2.7.0`

`rails ~> 6.1.4`

### Installation & using

In `terminal` or `cmd`

- `$ git clone git@github.com:XaoGao/Todoser.git`

- `$ gem install bundler` - if needed

- `$ yarn` - if needed

- `$ bin/webpack-dev-server` - if needed

- `$ bundle install`

- `$ rails db:migrate`

- `$ rails s` for server starting

Then open `http://localhost:3000` at your web-browser
